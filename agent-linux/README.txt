Endpoint Central — Linux Agent
================================

Cách sử dụng:
1. Mở file agent.conf, sửa EC_GATEWAY_URL thành IP của server
2. Chạy: sudo bash install.sh
3. Chọn [1] để test thử hoặc [2] để cài service thường trú

File trong package:
  install.sh  — Script cài đặt tự động
  agent.conf  — File cấu hình (chỉ cần sửa file này)
  README.txt  — File này
